# Landing Deployment

## Цель

Опубликовать лендинг проверки спроса в интернете, сохранив порядок проекта:

- исходники лендинга остаются в `src/web/landing/`;
- документация остается в `docs/`;
- деплой-конфигурация хранится в `.github/workflows/deploy-landing.yml`, потому что GitHub Actions требует этот путь.

## Рекомендуемый способ

Для текущего статического лендинга самый простой вариант - GitHub Pages.

Почему:

- не нужен backend;
- можно опубликовать HTML/CSS/JS напрямую;
- бесплатно для публичного репозитория;
- workflow уже настроен на публикацию только папки `src/web/landing`.

## Что уже подготовлено

- `src/web/landing/index.html` - страница лендинга;
- `src/web/landing/styles.css` - стили;
- `src/web/landing/app.js` - логика ссылки на Google Forms;
- `.github/workflows/deploy-landing.yml` - публикация лендинга через GitHub Pages.

## Что нужно для реальной публикации

1. Создать GitHub-репозиторий.
2. Загрузить проект в репозиторий.
3. Включить GitHub Pages с источником `GitHub Actions`.
4. Дождаться выполнения workflow `Deploy Landing`.
5. Получить публичный URL вида:

```text
https://<github-username>.github.io/<repo-name>/
```

## Важное ограничение формы

Лендинг не должен собирать заявки внутри HTML-формы и не должен сохранять контакты в `localStorage`.

Для публичного запуска нужно использовать Google Forms:

- создать форму по спецификации `docs/product/google-form-spec.md`;
- включить запись ответов в Google Sheets;
- вставить публичную ссылку формы в `GOOGLE_FORM_URL` в `src/web/landing/app.js`;
- не собирать ИИН, ФИО ребенка, точную дату рождения, диагнозы, медицинские документы и фотографии справок.

До подключения ссылки Google Forms кнопка на лендинге будет показывать сообщение, что форма еще не подключена.

## Минимальный следующий шаг

Выбрать один из вариантов:

- GitHub Pages - быстро и бесплатно, нужен GitHub-репозиторий;
- Tilda / Webflow / Framer - быстрее для no-code-редактирования, но придется перенести дизайн;
- Vercel / Netlify - удобно для frontend-проектов, нужна авторизация в сервисе.
